# live-beats-app
Module 2 pair project at Ironhack. Express App + mongoose

## User stories
- two kinds of users : page visior and bar owner

### Bar owner can:
* sign up 
* login / logout
* create event

### Page visitor can:
* filter events by type
* see event location on a map
* see an event details
